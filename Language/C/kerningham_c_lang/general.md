# General
- c는 상대적으로 저수준 언어이다.
- 정적변수 정의, 함수의 지역 변수에 의한 스택 취급 이외에 메모리 할당 기능을 언어에서 제공하지 않는다.
- 입출력 기능이 따로 없다.(read, write 등은 라이브러리 지원)
- 구조체 할당, 열거형 데이터
- 프리프로세서
- 명시적 형변환 필요
- 몇몇 연산자는 잘못된 우선순위를 가지고 있다.
- 1장 - c의 핵심적인 부분
- 2-6장 - c의 다양한 측면을 더 자세히
- 7장 - 표준 라이브러리
- 8장 - c, unix 운영체제 간의 인터페이스
- 부록 a - 언어 참조 메뉴얼
- 부록 b - 사용자를 위한 표준 라이브러리의 정의(ex: string)
- 부록 c - 최근의 업데이트
